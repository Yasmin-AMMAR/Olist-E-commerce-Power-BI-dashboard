# Olist-E-commerce-Power-BI-dashboard
Ce projet Power BI explore la performance commerciale d’Olist : analyse du chiffre d’affaires, cohorte clients, comportement des vendeurs, segmentation des retards de livraison et étude des catégories produits. L’objectif : identifier les leviers de croissance, optimiser l’expérience client et améliorer la qualité opérationnelle.
